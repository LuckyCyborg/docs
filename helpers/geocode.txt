# GeoCode

This function connects to google maps and retrieves the lat/long of the address provided

{line-numbers=off}
~~~~~~~~
GeoCode::getLngLat(['Hessle Road', 'Hull'])
~~~~~~~~

Returns:

{line-numbers=off}
~~~~~~~~
Array
(
    [lon] => -0.348839
    [lat] => 53.7388284
)
~~~~~~~~

Parameters are passed as an array there are 4 keys that can be used:

1. Street
2. City
3. State
4. zipcode
